# 阿凡提的豆子：指数增长的数学奥秘

## 一、故事内容

阿凡提的故事中有这样一个经典情节：

一位财主向阿凡提炫耀自己的财富，阿凡提说："如果你能答应我一个条件，我可以让你变得更富有。"财主好奇地问是什么条件。

阿凡提说："第一天你给我1颗豆子，第二天给我2颗，第三天给我4颗，第四天给我8颗……以后每一天给我的豆子数都是前一天的两倍，持续30天。"

财主心想：这算什么条件，每天才给这么点豆子，欣然同意了。

然而，当第30天到来时，财主才发现自己需要支付的豆子数量已经是一个天文数字……

## 二、对应的数学知识

### 1. 等比数列

这个问题本质上是一个**等比数列**问题：

- 首项 $a_1 = 1$
- 公比 $q = 2$
- 第 $n$ 天的豆子数：$a_n = 2^{n-1}$

### 2. 等比数列求和

前 $n$ 天的豆子总数可以用等比数列求和公式计算：

$$S_n = a_1 + a_2 + a_3 + \cdots + a_n = \frac{a_1(q^n - 1)}{q - 1} = 2^n - 1$$

### 3. 指数增长

这个问题的核心在于**指数增长**的威力：

| 天数 | 当天豆子数 | 累计豆子数 |
|:---:|:---:|:---:|
| 1 | 1 | 1 |
| 2 | 2 | 3 |
| 3 | 4 | 7 |
| 4 | 8 | 15 |
| 5 | 16 | 31 |
| 10 | 512 | 1,023 |
| 20 | 524,288 | 1,048,575 |
| 30 | 536,870,912 | 1,073,741,823 |

可以看到，前10天累计才1000多颗豆子，但到第30天，当天就需要支付5亿多颗豆子！

## 三、数学建模

### 问题分析

1. **输入**：天数 $n$（$1 \leq n \leq 30$）
2. **输出**：
   - 第 $n$ 天的豆子数
   - 前 $n$ 天的豆子总数

### 数学模型

- **当天豆子数**：$f(n) = 2^{n-1}$
- **累计豆子数**：$g(n) = 2^n - 1$

### 现实意义

指数增长在自然界和社会中广泛存在：
- 细菌繁殖
- 复利计算
- 病毒传播
- 计算机算力的摩尔定律

理解指数增长，有助于我们做出更明智的决策。

## 四、Python代码实现

```python
def beans_on_day(day):
    """
    计算第day天的豆子数
    参数: day - 天数
    返回: 第day天的豆子数
    """
    return 2 ** (day - 1)


def total_beans(day):
    """
    计算前day天的豆子总数
    参数: day - 天数
    返回: 前day天的豆子总数
    """
    return 2 ** day - 1


def print_bean_schedule(max_day=30):
    """
    打印豆子支付计划表
    参数: max_day - 最大天数，默认为30
    """
    print("=" * 50)
    print("阿凡提的豆子支付计划")
    print("=" * 50)
    print(f"{'天数':<8}{'当天豆子数':<15}{'累计豆子数':<15}")
    print("-" * 50)
    
    for day in range(1, max_day + 1):
        daily = beans_on_day(day)
        total = total_beans(day)
        print(f"{day:<8}{daily:<15,}{total:<15,}")
    
    print("=" * 50)


# 主程序
if __name__ == "__main__":
    # 打印完整的30天计划
    print_bean_schedule(30)
    
    print("\n关键数据：")
    print(f"第10天的豆子数：{beans_on_day(10):,}")
    print(f"第20天的豆子数：{beans_on_day(20):,}")
    print(f"第30天的豆子数：{beans_on_day(30):,}")
    print(f"前30天总豆子数：{total_beans(30):,}")
    
    # 换算成更直观的单位
    total = total_beans(30)
    print(f"\n换算：")
    print(f"约 {total / 10000:.0f} 万颗豆子")
    print(f"约 {total / 100000000:.2f} 亿颗豆子")
    
    # 假设一颗豆子重0.1克
    weight_kg = total * 0.1 / 1000
    weight_ton = weight_kg / 1000
    print(f"\n如果一颗豆子重0.1克：")
    print(f"总重量约 {weight_kg:.0f} 公斤")
    print(f"总重量约 {weight_ton:.0f} 吨")
```

# 再来一瓶：15%中奖率到底意味着什么

## 一、故事内容

小明去商店买饮料，瓶盖上有“再来一瓶”活动。

商家宣传：中奖率 15%。

小明就想：

1. 平均要买多少瓶，才能中到 1 瓶？
2. 如果连续买很多瓶，至少中 1 次的概率到底有多大？
3. 用 Python 模拟，结果会和理论接近吗？

这就是一个典型的**概率 + 编程模拟**问题。

## 二、对应的数学知识

### 1. 单次中奖概率

设每次买 1 瓶，中奖概率为：

$$p = 0.15$$

不中奖概率为：

$$1-p = 0.85$$

### 2. 买 n 瓶，至少中 1 次的概率

“至少中 1 次”更容易用反面事件计算：

- 反面事件：n 瓶都没中
- 概率：$(1-p)^n$

所以：

$$P(\text{至少中1次}) = 1-(1-p)^n = 1-0.85^n$$

例如：

- 买 1 瓶：$1-0.85^1=0.15$（15%）
- 买 5 瓶：$1-0.85^5\approx 0.5563$（约 55.63%）
- 买 10 瓶：$1-0.85^{10}\approx 0.8031$（约 80.31%）

### 3. 平均要买多少瓶才能中 1 次

这是几何分布的期望值：

$$E(X)=\frac{1}{p}=\frac{1}{0.15}\approx 6.67$$

含义：长期平均来看，大约买 6 到 7 瓶会中 1 次。

注意：

- 这不是说“第 7 瓶一定中”
- 只是大量重复实验后的平均值

## 三、数学建模

### 问题分析

1. 输入：中奖率 p（这里是 0.15）
2. 输出：
   - 理论上买 n 瓶至少中 1 次的概率
   - 单次实验“直到中奖”需要购买的瓶数
   - 重复多次实验后的平均购买瓶数

### 模型步骤

1. **如何模拟中奖率**
   - 使用 `random.random()` 生成 [0, 1) 的随机数
   - 若随机数 < 0.15，视为中奖

2. **单次循环抽奖，直到中奖**
   - 设 `count = 0`
   - 循环：每抽一次 `count += 1`
   - 若中奖则停止，返回 `count`

3. **循环 10 次，统计每次抽奖次数并求平均**
   - 重复第 2 步 10 次
   - 记录每次所需抽奖次数
   - 计算平均值

## 四、Python 代码实现

```python
import random


def theoretical_at_least_one_win(n, p=0.15):
    """
    理论计算：买 n 瓶至少中 1 次的概率
    """
    return 1 - (1 - p) ** n


def draw_once(p=0.15):
    """
    单次抽奖：返回是否中奖
    """
    return random.random() < p


def bottles_until_first_win(p=0.15):
    """
    单次实验：不断购买，直到第一次中奖
    返回购买瓶数
    """
    count = 0
    while True:
        count += 1
        if draw_once(p):
            return count


def run_trials(trials=10, p=0.15):
    """
    重复 trials 次“直到中奖”实验
    返回每次结果和平均购买瓶数
    """
    results = []
    for _ in range(trials):
        results.append(bottles_until_first_win(p))
    average = sum(results) / len(results)
    return results, average


if __name__ == "__main__":
    p = 0.15

    print("=" * 50)
    print("再来一瓶中奖率分析")
    print("=" * 50)

    print("\n一、理论概率（买 n 瓶至少中 1 次）")
    for n in [1, 3, 5, 10, 20]:
        prob = theoretical_at_least_one_win(n, p)
        print(f"买 {n:>2} 瓶 -> 至少中1次概率: {prob:.4f} ({prob * 100:.2f}%)")

    expected = 1 / p
    print("\n二、理论平均购买瓶数")
    print(f"中奖率 p = {p:.2f}")
    print(f"平均需要购买约 {expected:.2f} 瓶才能中 1 次")

    print("\n三、模拟实验（循环 10 次，每次直到中奖）")
    results, avg = run_trials(trials=10, p=p)
    for i, cnt in enumerate(results, start=1):
        print(f"第{i:>2}次：买了 {cnt:>2} 瓶才中奖")
    print(f"10次平均购买瓶数：{avg:.2f}")

    print("\n说明：")
    print("- 单次购买的中奖概率始终是 15%，不会因为前几次没中而自动提高。")
    print("- 若采用“直到中奖”为止的玩法，最终中奖概率是 100%，但可能要买很多瓶。")
```

## 六、总结

1. **理论中奖率**：单次购买中奖率固定是 15%。
2. **买 n 瓶的中奖率**：至少中 1 次的概率是 $1-0.85^n$。
3. **平均成本直觉**：长期看，约买 6 到 7 瓶中 1 次。
4. **模拟的价值**：通过编程可直观看到随机性，理解“平均值不等于每次都一样”。

这个例子告诉我们：**概率能指导决策，编程能验证直觉**。理论与实验结合，才能真正理解“15%中奖率”背后的含义。
